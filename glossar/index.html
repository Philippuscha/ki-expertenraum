KI-Glossar: Alle Begriffe einfach erklärt

Die wichtigsten KI-Begriffe verständlich erklärt – für Anfänger und Fortgeschrittene

A

AI (Artificial Intelligence)

Künstliche Intelligenz – die Fähigkeit von Maschinen, menschenähnliche Intelligenz zu zeigen: zu lernen, Probleme zu lösen und zu verstehen.

API

Application Programming Interface – eine Schnittstelle, über die Programme miteinander kommunizieren können.

C

ChatGPT

Ein KI-Chatbot von OpenAI, der natürliche Sprache versteht und generiert.

Context Window

Das "Gedächtnis" einer KI – wie viel Text sie gleichzeitig verarbeiten kann.

D

Deep Learning

Fortgeschrittenes Machine Learning mit vielen Schichten neuronaler Netze.

G

Generative KI

KI, die neue Inhalte erzeugt – Texte, Bilder, Musik oder Videos.

L

LLM (Large Language Model)

Großes Sprachmodell wie GPT-4 – trainiert auf riesigen Textmengen.

M

Machine Learning

Maschinelles Lernen – Systeme lernen aus Daten, ohne explizit programmiert zu werden.

P

Prompt

Deine Eingabe an die KI – die Anweisung, was sie tun soll.

Prompt Engineering

Die Kunst, optimale Prompts zu formulieren für beste Ergebnisse.

T

Token

Die kleinste Einheit in Sprachmodellen – etwa ein Wort oder Wortteil.

Transformer

Die Architektur hinter modernen KI-Modellen – revolutioniert durch "Attention"-Mechanismus.
