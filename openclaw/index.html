OpenClaw Installation & Konfiguration

Der komplette Guide: Von der Installation bis zur produktiven Nutzung mit maximaler Datensicherheit

Was ist OpenClaw?

OpenClaw ist eine Open-Source-Plattform für lokale KI-Modelle. Die Alternative zu ChatGPT mit voller Datenkontrolle.

Ideal für Unternehmen, die maximale Datensicherheit und Unabhängigkeit von Cloud-Anbietern wollen.

Voraussetzungen

  • Hardware: 4 Kerne CPU, 8GB RAM, 50GB Speicher
  • Software: Linux/Ubuntu oder Windows mit Docker
  • Netzwerk: Stabile Internetverbindung

1Repository klonen

git clone https://github.com/openclaw/openclaw.git
cd openclaw

2Konfiguration anpassen

cp config.example.yml config.yml
nano config.yml

Wichtige Einstellungen:

server:
  port: 3000
  host: 0.0.0.0

ai_models:
  default: llama2
  available:
    - llama2
    - mistral
    - codellama

3Docker starten

docker-compose up -d
Hinweis: Der erste Start lädt die KI-Modelle herunter. Das kann je nach Internetverbindung 10-30 Minuten dauern.

4Admin erstellen

docker exec -it openclaw python manage.py create_admin

5Testen

Öffnen Sie http://localhost:3000 im Browser und melden sich mit dem Admin-Account an.

Kosten-Nutzen-Rechnung

  • Einmalkosten: Server 2.000-5.000€
  • Laufende Kosten: ~50€/Monat (Strom)
  • Einsparungen: Keine ChatGPT-Lizenzen (500€/Monat)
  • Amortisation: 6-12 Monate
